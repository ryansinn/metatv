## ЁЯФз Findings (prioritized)

### P1 тАФ `session.expunge` тЖТ DTO  *(Band 9 B9-2, still open)*
**12 sites** in `main_window_metadata.py` + `main_window_favorites.py` (grew from 9 тАФ the new
`play_channel_new_window_by_id` added one). The CLAUDE.md "ORM must not outlive its session"
rule names `expunge` the *fragile fallback*, safe only while the ORM stays relationship-free.
Build a `PlayableDTO` (+ `PlayableEpisodeDTO`) inside the `session_scope` and drop the expunge.
Detailed plan: `docs/REFACTOR_PLAN_BAND9.md` ┬з B9-2.

### P1 тАФ render-time `parse_channel_name` drift
**9 GUI sites; only 1 is the documented exception** (`epg_view._make_recommendation_item`). The
rest violate the ingestion-only rule (read stored `detected_*`, don't re-parse at render):
- `gui/details_sections.py:360` (`load_basic`), `:460` (`load_metadata` тАФ parses *metadata* title; borderline)
- `gui/details_similar.py:116`
- `gui/sidebar/base.py:23`
- `gui/main_window.py:2568`
- `gui/epg_view.py:1639`, `:1792` (watchlist / My-Channels render)

The EPG On-Now fix (#53) addressed only one spot; this is the same stored-field pattern applied
to the rest. Each needs the consuming surface to read `detected_prefix`/`detected_title`/etc., or
a DTO carrying them (some, like `details_similar`, already have a DTO that could carry the fields).

### P2 тАФ files over 1000 lines
`epg_view.py` **2959**, `main_window.py` **2910** (both *grew* during the feature push),
`provider_editor.py` **1291**, `channel.py` **1016**. The long-deferred decomposition
(B7-7/8). Candidate splits: EPG tab modules (Watchlist/OnNow/Browse/Events/Discover); a
MainWindow provider-toggle / channel-load orchestration mixin; `channel.py` query split.

### P2 / rules-alignment тАФ the `px` half of the theme rule is not enforced
0 hex, but **133 `font-size: Npx`** literals bypass the existing `FONT_*` tokens, plus **61
structural `padding`/`border-radius` px** that have *no* token. The rule as written ("no px
literal") does not match reality ("colors always tokenized; sizes often inline"). **Resolve the
rule first** (see Band 10 item B10-3), then a gradual `font-size`тЖТ`FONT_*` cleanup. Worst files:
`epg_view.py` (16), `similar_lightbox.py` (15), `global_filter_dialog.py` (11),
`categories_dialog.py` (10).

### P3 тАФ long tail (low urgency)
- **`get_session()` тЖТ `session_scope`:** 76 sites / 23 files. Rule already permits legacy; migrate
  opportunistically. (New EPG-relink code uses `get_session()` to match the surrounding
  `EpgManager` pattern тАФ acceptable, noted.)
- **Band 9 B9-1** (`load_channels` тЖТ `_run_query` seam) тАФ still open; needs a display smoke-test.
- **`executor.submit` outside the seam:** ~22 sites тАФ mostly legit (seam internals, EPG/image
  workers); a per-site pass could fold a few into `_run_query`. Low value.
